# SafeSignal

SafeSignal is a web app that helps users detect scam messages, phishing attempts, suspicious links, and social engineering patterns.

## Features
- Detects scam-like phrases and phishing indicators
- Analyzes suspicious links
- Gives a risk score
- Explains why a message looks suspicious
- Includes built-in demo examples
- Saves recent scan history in the browser
- Lets users copy analysis results

## Why I Built This
Scams, phishing, and fake job offers are everywhere. I wanted to build a practical tool that helps everyday users quickly analyze suspicious messages and understand warning signs.
